# Beer Suggestion Program

An interactive Python application that pairs beer styles with food based on professional brewing guides.

## Features
- Extensive database including Lagers, Pilsners, Ales, IPAs, and Stouts.
- Tracks ABV, IBU, and recommended serving temperatures.
- Direct food-to-beer pairing logic based on the "Ultimate Guide to Beer Styles."

## How to Use
1. Run the script: `python "Suggest_a_beer.py"`
2. Enter your name.
3. Choose a food category from the list to receive a recommendation.
4. Follow the prompts to see specific examples or beer metrics.
